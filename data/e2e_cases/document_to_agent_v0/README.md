# Document-to-Agent E2E v0

This case starts from `input_material.md` and checks whether the system can derive bounded Skill and Knowledge artifacts, build a ReleaseBundle, bind it, inject it into a runtime/backend, execute one repo-level dependency-use triage task, and emit verifier/report artifacts.

This is a local development E2E validation case. It is not an official public benchmark and does not claim production readiness.
