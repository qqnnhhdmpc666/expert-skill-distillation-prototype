# Demo Script Template

## Demo Goal

Show that the system can distill expert materials into inspectable, correctable, reusable agent skill packages.

## Demo Flow

### 1. Input Materials

Show:

- expert document;
- example task;
- workflow;
- correction record.

Clarify:

- data is public / synthetic / anonymized;
- no sensitive materials are used.

### 2. Distillation

Show generated:

- capability track;
- behavior track;
- constraints;
- examples;
- verifier checklist.

### 3. Verification

Show:

- evidence-supported items;
- unsupported items;
- contradictions;
- items needing correction.

### 4. Feedback Correction

Example feedback:

```text
This rule is too broad; restrict it to the review scenario.
```

Show:

- skill v1;
- correction;
- skill v2;
- version log.

### 5. Efficiency Mode

If implemented:

- full skill package;
- compact invocation package;
- token/call comparison;
- checklist pass retention.

## Final Message

The system does not replace experts. It turns expert materials into inspectable, correctable, reusable agent skills and prepares them for future migration or lower-cost invocation.
